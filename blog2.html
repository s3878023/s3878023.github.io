BASIC GIT COMMAND.

To able to use Git and GitHub, there are a few simple git commands to remember:

-Configure the author name and email address to be used with your commits. here are the examples: git config --global user.name "anonymuser" git config --global user.email anonymuser@example.com

git clone Usage: git clone [url] This command is used to obtain a repository from an existing URL.

>

git add Usage: git add [file] This command adds a file to the staging area.
